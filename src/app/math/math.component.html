Sets: Rule of Sums

A set A with k elements, a set B with n elements, guaranteed that these sets do not have common elements. Then, the rule of sum states that A union B contains n plus k elements. By means of a Venn Diagram we can view sets. Important: in the rule of sum, no object should belong to both classes!

Python Tools:
  • Unordered with Repetitions (Combinations w/ Repetitions
    from itertools import combinations_with_replacement
    count = 0
    for c in combinations_with_replacement(['a ','b ','c '], 4):
    print("".join(c))
    count +=1
    print(count)
  • Unordered without Repetitions (Combinations w/o Repetitions)
    from itertools import combinations
    for c in combinations("abc", 2):
    print("".join(c))
  • Permutations
    from itertools import permutations
    for c in permutations("abc", 2):
    print("".join(c))
  • Tuples
    from itertools import product
    for c in product("abc", repeat=2):
    print("".join(c))

A Linked List is a collection of nodes which together represent a sequence. Each element's location in memory is trivial due to the fact that each element(or node) contains a pointer to the next node in the sequence. This maintains some order, or more acurately a way to search the list for common tasks such as insert, remove, find, etc. The tool below, created dynamically with d3.js, helps portray a linked list's inner workings.
